# 前提条件
`docker`、`docker-compose`はインストール済とする

# 環境構築
当リポジトリをclone後、`docker-compose.yml`が配置されたディレクトリで`docker-compose up`を実行

# ローカルサーバー
webpack-dev-serverで起動したWebサーバー（ホットリロード対応）
